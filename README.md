# BandForce

Premium resistance bands and fitness accessories for serious athletes and fitness enthusiasts.

## About BandForce

BandForce is a full-featured eCommerce platform specializing in high-quality resistance bands and workout accessories.

**Visit us:** https://bandforce.fly.dev

## Technology

BandForce is built with modern, production-ready technology:

**Frontend:** React 18, Redux, Tailwind CSS, React Router
**Backend:** Node.js, Express, MongoDB
**Payments:** PayPal integration
**Hosting:** Deployed on Fly.io

## Features

- Complete eCommerce experience with product browsing and search
- Persistent shopping cart
- User authentication and profile management
- Real-time order tracking
- Secure PayPal checkout
- Mobile-responsive design
- High-quality product images
- Organized product categories
- Dynamic shipping and tax calculation
- Admin dashboard for store management
